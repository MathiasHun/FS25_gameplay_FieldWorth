## Description
This mod dynamically calculates farmland prices based on its current state. 
It considers plowing, rolling, liming, fertilization, and harvest readiness to adjust the price. 
Currently, crop type and weed levels are not factored in, but these features may be added in future updates.

### In its current form, the mod shows:
  - Base price of the field
  - Price changes due to maintenance or neglect
  - Total effect on the field's value

### Future updates may include:
  - A settings menu to adjust bonus/penalty percentages
  - Consideration of crop types and weed levels for more precise valuation

This mod enhances realism by making field management directly affect land value.

Bug report, please visit https://farmsim.bltfm.hu/ or https://github.com/MathiasHun/FS25_gameplay_FieldWorth/

This is a beta version intended for testing and collecting feedback.
I welcome any bug reports, comments, or suggestions.
This mod has not yet been tested in multiplayer (server) environments.
Thank you for taking the time to read this.

## Description
Ce mod calcule dynamiquement le prix des terres agricoles en fonction de leur état actuel.
Il prend en compte le labour, le roulage, le chaulage, la fertilisation et l’état de préparation à la récolte pour ajuster le prix.
Actuellement, le type de culture et le niveau de mauvaises herbes ne sont pas pris en compte, mais ces éléments pourront être ajoutés dans de futures mises à jour.

### Dans sa version actuelle, le mod affiche :
  - Le prix de base du champ
  - Les variations de prix dues à un bon entretien ou à la négligence
  - L’impact total sur la valeur du champ

### Les futures mises à jour pourront inclure :
  - Un menu de réglages pour ajuster les pourcentages de bonus et de pénalités
  - La prise en compte des types de cultures et des mauvaises herbes pour une évaluation plus précise

Ce mod améliore le réalisme en faisant dépendre directement la valeur du terrain de sa gestion.
